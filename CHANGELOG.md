# Changelog

All notable changes to `sweepr` are documented here.

## v0.1.1 - 2026-05-14

- Added `sweepr types` to list supported file categories and extensions.
- Added dry-run category summaries showing planned files per destination folder.
- Added `--exclude` support for skipping paths and glob patterns.
- Expanded supported file categories and extensions.

## v0.1.0 - 2026-05-14

- Initial open-source release.
- Added safe file organization by type or modification date.
- Added dry-run previews, recursive scanning, undo manifests, Rich output, tests, and CI.
